Shorthelp

Executing "Startdelayed.exe" without any startparameter
runs all defined StartCmds, besides those that have the modifier "inactive".

Startparameter

No parameter -> run all defined StartCmds
"stopall" -> run all defined StopCmds
"editmode" -> open the list of commands

Supplied helper

"startdelayedStopall.exe" -> run "Startdelayed.exe stopall"
"startdelayedEdit.exe" -> run "Startdelayed.exe editmode"

Editmode

Click on a row -> execute the StartCmd
(entries with modifier "inactive" can be executed too!)
[Ctrl] + click on row -> execute the StopCmd
[Shift] + click on a row -> Edit row
All entries are saved in the "Directories-file" -> "stddirectories_.txt" as comma-separated values (CSV),
also editable via the "menu" -> "Setup" -> "Edit Directories-file: stddirectories_.txt".
